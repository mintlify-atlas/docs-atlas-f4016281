# /parallel Command

Set up parallel Claude Code sessions using git worktrees.

## Native Worktree Mode

Claude Code 2.1.49+ has built-in worktree support:

```bash
claude --worktree
claude -w
```

This creates an isolated git worktree automatically.

See the [Parallel Worktrees concept](/concepts/parallel-worktrees) and [skill](/skills/parallel-worktrees) for complete details.
