# /list Command

Display all learnings stored in the pro-workflow database.

## Syntax

```bash
/list
/list recent
/list category:Testing
/list project:my-app
```

## Output Format

```text
Pro Workflow Learnings (15 total)

Recent learnings:
#42 [Navigation] Confirm full path when multiple files share a name
    Created: 2 days ago
    Project: my-app
```

See [Database Learnings API](/api/database/learnings) for complete details.
