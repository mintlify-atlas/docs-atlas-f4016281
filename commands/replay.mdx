# /replay Command

Automatically find and surface relevant learnings from your pro-workflow database.

## Syntax

```bash
/replay <task description>
```

**Examples:**
```bash
/replay auth middleware
/replay "file path errors"
/replay testing react components
```

See the [Replay Learnings skill](/skills/replay-learnings) for complete details.
